# Garabato Store

E-commerce de streetwear contemporáneo construido con **React + Vite + Tailwind CSS v4 + Framer Motion + Zustand**.

## Stack
- React 18 · Vite 5
- React Router 6
- Tailwind CSS v4 (con tokens propios)
- Framer Motion (animaciones)
- Zustand (estado: carrito, UI, filtros)
- Lucide React (iconos)
- @fontsource (Bebas Neue + DM Sans)

## Desarrollo
```bash
npm install
npm run dev          # http://localhost:5173
npm run build        # output → dist/
npm run preview
```

## Deploy
Conectado a Vercel — cada push a `main` redespliega.
